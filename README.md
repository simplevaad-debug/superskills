## What Are Skills?

Skills are instruction files that teach Claude Code **how** to be an expert. Instead of getting generic AI responses, you get output from a specialist who knows the frameworks, patterns, and constraints of your domain.

**Without skills:** "Here's a basic React component..."

**With `react-expert`:** Production-grade React 19 with Server Components, proper TypeScript, error boundaries, accessibility, and testing — following the exact patterns senior engineers use.

## How Skills Work

Each skill has:

```
skills/skill-name/
├── SKILL.md           # Main instructions (what the skill knows)
└── references/        # Deep-dive docs loaded on demand
    ├── topic-a.md
    └── topic-b.md
```

When you ask Claude Code to build a React component, `react-expert` activates automatically and provides:
- **Expert patterns** — Server Components, hooks, performance optimization
- **Constraints** — What to do and what to avoid
- **Code templates** — Production-ready examples
- **Reference docs** — Loaded on demand for deep topics
