## What Are Claude Code Skills?

**Claude Code skills are structured instruction files (`.md`) that give Claude Code domain-specific expertise.** Each skill teaches Claude the frameworks, patterns, constraints, and best practices for a specific area — turning generic AI responses into production-grade, expert-level output. Skills are installed in `~/.claude/skills/` and activate automatically based on context.

### Without Skills vs. With Skills

| | Without Skills | With `react-expert` Skill |
|---|---|---|
| **React output** | Basic component with useState | React 19 Server Components, proper TypeScript, error boundaries, accessibility, Suspense, testing |
| **Quality** | Tutorial-level | Senior engineer-level |
| **Production-ready?** | No | Yes |

## How Skills Work

Each skill is a folder with a `SKILL.md` file:

```
skills/skill-name/
├── SKILL.md           # Main instructions (what the skill knows)
└── references/        # Deep-dive docs loaded on demand
    ├── topic-a.md
    └── topic-b.md
```

When you work on a task, Claude Code detects the relevant skill and applies:
- **Expert patterns** — Domain-specific best practices and architecture
- **Constraints** — What to do and what to avoid
- **Code templates** — Production-ready examples
- **Reference docs** — Loaded on demand for deep topics

## FAQ

### Are Claude Code skills free?
Some are. This repo includes 5 free, open-source skills. The full 139-skill collection is $50 one-time at [superskills-one.vercel.app](https://superskills-one.vercel.app).

### Do skills work with all Claude Code versions?
Yes. Skills are a core feature of Claude Code and work across all versions.

### How many skills can I install?
There's no limit. All 139 skills can be active simultaneously — Claude picks the right one based on context.

### What's the difference between skills and MCP servers?
Skills teach Claude **how** to approach a domain (patterns, best practices, constraints). MCP servers give Claude access to **external tools and data** (APIs, databases, services). They complement each other.
